typedef struct bgc_Allocation bgc_Allocation

The allocation object.

The allocation object holds all metadata for a memory location in one place.

◆ bgc_AllocationMap

typedef struct bgc_AllocationMap bgc_AllocationMap

The allocation hash map.

The core data structure is a hash map that holds the allocation objects and allows O(1) retrieval given the memory location. Collision resolution is implemented using separate chaining.
